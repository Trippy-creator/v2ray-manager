# V2Ray 全能管理脚本



> 支持全协议、多用户的 V2Ray 配置管理工具

## 功能特性

### 协议支持
- ✅ VMess/VLESS/Trojan/Shadowsocks
- ✅ Socks5/HTTP/MTProto/DNS
- ✅ WireGuard (需内核支持)

### 核心功能
- 🚀 一键安装/卸载/升级
- 🔧 交互式配置向导
- 📊 流量统计与限制
- 🔄 多版本管理与回滚
